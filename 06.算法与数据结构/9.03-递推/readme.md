## 递推问题求解套路
  1. 确定递推状态
    
    一个函数符号f(x)，外加这个函数符号的**含义描述**
    
    通常这个函数所对应的值，就是要求解的值
    
    注：f(x) = y; x = 自变量，y = 因变量

  2. 确定递推公式
  3. 分析边界条件
  4. 程序实现
