# 基于cra的项目模板

内置
- cra 
- redux
- router
- antd
- proxy
- swiper
- echart
- 兼容ie9

### 安装
  `npm i eol-template -g`

### 创建项目
  `eol create`

### 选择项目模板，填写项目名称
  ```javascript
    ? What's your project type: pc
    
    ? What's your project name: my-app
  ```

### 跳转项目目录
  `cd my-app`

### 安装依赖
  `yarn`

### 启动
  `yarn start`
