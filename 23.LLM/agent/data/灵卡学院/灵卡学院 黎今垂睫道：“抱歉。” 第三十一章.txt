这公告一出，校论坛讨论得热火朝天。老生们羡慕极了：“这届新生也太好运了吧！六星卡环啊啊啊！下半辈子不愁了！”“虽然但是，能拿第一的还缺个六星卡环吗？”“也对。世家一来不差钱，二来早就给本家孩子养了本命卡环。”“即便拿不到卡环，只要入围就能学分全满评级 A+啊！”“呜呜呜为什么只有新生杯，这不公平，等一个老生杯！”

新生们讨论得就更加热烈了。“要不要搏一搏！”“博个鬼啊，看清楚条件，如果真能入围，你还怕学分不满外加年评级不够吗？”“只能使用自己制作的卡片，还只有四个名额，呵呵，提前考试罢了！”

卢弦乐和林听早就知道了新生杯的事，宁烛在周末就告诉她们了。卢弦乐私下里跟林听说：“这听着就不是我们能参加......”林听却道：“为什么不能？”卢弦乐：“只有四个名额......”林听瞪她：“不入围也无所谓啊，这是不可多得的试炼机会！”卢弦乐：“啊这......”林听道：“现在不抓住机会试炼，两年后的卡师‘奥运会’你也想只当个观众吗？况且，校长早告诉我们了，危险近在眼前，我们得抓住一切机会强大自己！”卢弦乐：“！”

宁烛从李素文的办公室出来后，就回泡泡乳茶店找到了黎乐洋。她把卡环还给了他，也和他说了新生杯的事。黎乐洋一整个精神抖擞，道：“我这就搞个赛前分析，到时候直接发论坛，想参加选拔的都可以冲一波，这回咱们灵卡学院必须拿第一！”不愧是白羊系的代表学生。乐子羊的热血和激情，笼罩全校！

周一的公告一出来，乐子羊就发帖了。@无限好文，尽在晋江文学城。帖子一出，新生们又又又沸腾了。宁烛一点进去，先看到的是新生们的磕头表情包。卢弦乐和林听也都看到了帖子，她们直直倒吸气：“黎同学也太靠谱了！”何止是靠谱，关键是大方。他发的全是干货，是真不藏私啊！不只是白羊系的新生们涌了进来，就连天秤系的也匿名摸进来，开始仔仔细细研究起来了。

黎乐洋分析了前几届卡师大赛的赛制，给出了新生杯赛制的大体推测。最关键的肯定是擂台赛，也就是卡师 1V1，最后只能决出唯一的冠军。其次是双人赛，卡师 2V2。一般情况下是系内组队，也有可能是系外组队——极其少见，几十年能见一次。新生杯想必也是大差不差，基本就是这两个比赛模式。

比赛的内容简单粗暴。在随机或指定【场景卡】中，卡师进行 1V1 或者 2V2 的对决。获胜条件则是看【角色卡】状态。一方卡师将另一方卡师的【角色卡】彻底摧毁，且失去【角色卡】的一方没有可用卡片，则摧毁方获胜。如果两位卡师的【角色卡】同时被摧毁，那就看手中卡片的剩余情况，剩余越多的（包括剩余使用次数）的一方获胜。如果卡师的卡片也同时用空了，则需要双方卡师进行近身搏斗，击倒对方后获胜。

卢弦乐看着比赛内容，喃喃道：“有够刺激的。”林听：“想想开学第一课。”卢弦乐不吱声了，她无比清楚地记起了杨校长的话——卡师既是战士。

比赛终究是比赛。为了确保卡师们的安全，进行了很多规则限制。如果是现实中，卡师对决可不会只攻击【角色卡】。就像当初在地下停车场时，黎今甚至没用过【角色卡】。

这时，又有一个帖子被顶了上来，发帖人的 ID 是“神官”。这 ID 跟乐子羊一样，真是明眼人一看就知道是谁。宁烛对沈观没什么好印象，主要是觉得和他气场不和。

他也发了一个“科普帖”，但和黎乐洋的截然不同。黎乐洋的帖子在写满干货的同时，鼓励大家勇敢参加校内选拔，字里行间都是热血激情，中二之气喷薄欲出。看他帖子的新生们也的确被煽动了情绪，好像立刻马上就能大战三百回合，冲进新生杯入围赛，为灵卡学院拿下冠军，为华夏争光夺彩。

沈观的帖子则是瞬间让人看清现实，他不愧是天秤系的佼佼者，出手就是清晰的表格，罗列了参加新生杯选拔需要的支出和风险。【角色卡】至少三张，需要自己制作，其中的制卡材料的成本在六百到六千（基本可以确定，一年级新生做不出三星角色卡）。【术法卡】至少十张，需自己制作，其中制卡材料的成本在一千到一万。【装备卡】至少需要三张，需自己制作，其中制卡材料的成本也得是六百到六千。更不要提还需要备好【灵能卡】......

卡片成本高也就算了，重点是得自己做。一年级新生的制卡成功率平均在 10%左右，那么材料成本就得再翻上十倍。这一条条给出来，很多新生都萎了。热血沸腾是需要金钱支持的啊！

沈观后续还给了一个这学期的制卡课课程。一年级生能学的制卡知识有限，主要是以启蒙和了解为主，基本上也就系统学习一星【灵能卡】和一星【角色卡】的制作。【术法卡】要下学期才会接触，【装备卡】更是要到二年级才会教授。也就是说，想要参加新生杯，他们需要自学如何制作【术法卡】和【装备卡】。

黎乐洋的帖子里大家热血沸腾。沈观的帖子里则是一片唉声怨气。有白羊系学生的在沈观帖子里反驳道：“校公告里说了，比赛中损失的卡片，学校都会给与材料补偿！”“神官”特意置顶了这条回复，并在下面说道：“首先，赔付的只是单次材料；其次，如果真能做出这些卡片，转手就是翻倍的收入。”还是有人不服：“少说丧气话！我看你就是不想让大家报名参赛，这样自己才有机会入围三院新生杯！”帖子里立马有人讽刺道：“楼上一看就是白羊的蠢货，楼主好心提醒，不领情也就算了，还反咬一口，你知道楼主是谁吗？他百分百入围好吧！”“少地图炮，什么叫白羊的蠢货？就你们天秤的都是大聪明呗！”“我说错了吗，你看他 ID，百分百白羊系。”“白羊系怎么了，你们天秤系傻缺更多！”

帖子更加火爆了，不是因为讨论新生杯，而是两个系互喷，逐渐失去理智。

宁烛看完沈观的帖子，倒是对他改观不少。他说得挺客观的，从更加现实的角度分析了新生杯的情况。一年级新生也就十六七岁，正是冲动热血的年纪，想做什么就做什么，很容易冲动行事。回头浪费了一堆钱，也没有入围比赛，还耽误了该学习的课程，的确是得不偿失。

包括卢弦乐和林听，也都在重新考虑参赛的事了。这的确是比不小的费用......

让大家出乎意料的是，下午又出了一个校公告，对新生杯进行了更加周全的补充说明。其中一条最为瞩目——所有报名参赛的学生，都可以去校仓库免费申领制卡材料。但是，学生们需在指定的制卡实验室制卡，制成的卡片也只能用于新生杯的比赛，如果卡片没被使用就被淘汰，校方会回收相应的卡片。同时，校方还着重强调，一旦入围十二月的三院对战，则会免费提供所有制卡材料，只要学生懂得制卡原理，哪怕制卡成功率仅有 1%，也会予以材料支持。

这公告一出，熄火的新生们又斗志昂扬了。之前在沈观的帖子里被骂的人又蹦跶了几下，但很快就被群嘲了。@无限好文，尽在晋江文学城 “别黑沈观音了，这波真没得黑。”“要不是他这帖子，校方也不会给出这补充说明。”“本白羊原谅天秤一秒钟，沈观音好样的！”“谁稀罕你们原谅啊，以及什么沈观音啊，我们沈爷是‘神官’！”

于是，帖子里又又骂了起来。校论坛的管理员见怪不怪，反正删也删不完，只要没越线就当没看见了。

周一这一整天，哪怕在课上，新生们也忍不住讨论关于新生杯的事。教授们不得不一次次点名，让他们集中精神好好听课！入围的只有四个人，其余人可没有学分全满的优待！

宁烛一整天都心不在焉，她也关注新生杯，但在看了两个帖子之后就退出了校论坛。@无限好文，尽在晋江文学城 她肯定会参加，也会为此好好准备。制卡是门学问，卡战更是需要学习和练习。再加上她的卡环不适配，还得额外学习压制灵能......幸好比赛在十二月，否则她别说夺冠了，入围都难。

宁烛是因为晚上的补课而心神不宁。今天是周一，她晚上七点要去江冰轮的办公室，跟他学习天秤序列的专业课。只是想一想，宁烛都觉得头大。一对一补课已经够可怕了，还要跟一个讨厌自己的老师补课，真是......可怕他爹给可怕开门，可怕到家了！

伸头一刀，缩头也是一刀，宁烛在吃过晚饭后又精神抖擞了。她跟爸妈视频完之后，雄赳赳气昂昂地去了天秤系主任的办公室。有什么好怕的？江主任只怕比她更头大吧！这么一想，宁烛反倒舒坦了。

宁烛提前五分钟就到了，她不想提前进去，所以就在楼道里等着，想着最多提前一分钟再去敲门。就在这时，江冰轮的办公室门开了，宁烛心猛地提起。早知道就晚点上楼了！这要是撞上了，是不是要提前上课？宁烛心中念头刚转了一圈，就看清了推门而出的人。不是江冰轮，而是黎今。

自从上次【命运转换】后，他们没再见过面。宁烛冷不丁看到他还愣了下。“黎学长！”她赶紧打招呼。黎今低垂着眼睫，似乎在想什么，听到她的声音才微微抬眸，一双深色的眸子落在她身上。宁烛：“......”他肯定刚用过【天秤卡】，眼中还残余着那奇妙的瑰丽光泽！

“我听说你的卡环因灵能超载爆炸了。”黎今看向她手腕道。宁烛努力和他保持距离：“已经没事了。”黎今垂睫道：“抱歉。”宁烛：“？？？”她半晌才反应过来。当初在灵韵之地，黎今去追盗窃七星卡环的人，结果却被困在【迷宫卡】里。他是在为自己没有追回七星卡环而道歉。

宁烛哭笑不得道：“黎学长，你和李教授说了同样的话，可这只是一场意外，和谁都没关系的。”“况且我从不觉得那枚七星卡环属于我。”她顿了下，看向黎今道：“比起那枚卡环，我更想赢下一枚真正属于自己的卡环。”
